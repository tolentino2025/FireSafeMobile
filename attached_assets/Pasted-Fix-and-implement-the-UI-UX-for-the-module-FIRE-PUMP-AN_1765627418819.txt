Fix and implement the UI/UX for the module
"FIRE PUMP ANNUAL PERFORMANCE TEST – DIESEL ENGINE FIRE PUMP"
based strictly on NFPA 25 Handbook.

==================================================
GLOBAL UI REQUIREMENTS (MANDATORY)
==================================================
- Use a SINGLE VERTICAL SCROLL SCREEN (mobile-first).
- DO NOT use accordion, collapsible sections, dropdown sections, or expand/collapse arrows.
- ALL sections must be visible in sequence while scrolling.
- Use section headers and card-style blocks to visually separate content.
- Keep all NFPA 25 labels EXACTLY in English (no renaming, no translation).

==================================================
ALLOWED DROPDOWNS (PRE-REGISTERED DATA ONLY)
==================================================
Dropdowns are allowed ONLY for selecting pre-registered entities:

- Diesel Fire Pump
- Contractor Company
- Inspector
- Owner / Property Owner
- Contracting Party / Client
- Job / Site

When a dropdown value is selected:
- Auto-populate all related fields in the form.
- Allow override ONLY for this test record.
- NEVER overwrite master records unless user explicitly clicks "Update Master Record".

==================================================
SECTION 9 — PERFORMANCE VERIFICATION (YES / NO / N/A)
==================================================
Include all NFPA verification items exactly:

- G. Ampere demand ≤ full load amperes × service factor
- H1. Actual voltage × current ≤ rated values × service factor
- H2. Voltage ≤ 5% above rated
- H3. Voltage ≤ 10% above rated
- I. Did engine-driven units operate without any signs of overload or stress?
- J. Was the engine overspeed emergency shutdown tested?
- K. Was the governor set to properly regulate the engine speed at rated pump speed?
- L. Did the gear drive assembly operate without excessive objectionable noise, vibration, or heating?
- M. Was the fire pump unit started and brought up to rated speed without interruption under peak load?
- N. Did the fire pump performance equal the manufacturer’s factory curve?
- O. Did the electric motor pumps pass phase reversal test on normal and alternate power?
