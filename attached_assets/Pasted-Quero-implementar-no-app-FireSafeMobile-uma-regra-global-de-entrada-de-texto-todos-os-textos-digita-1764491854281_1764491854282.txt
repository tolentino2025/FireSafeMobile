Quero implementar no app FireSafeMobile uma regra global de entrada de texto:
todos os textos digitados pelo usuário devem ser automaticamente convertidos para MAIÚSCULAS, exceto campos de e-mail.

Implemente as seguintes alterações:

1) Criar um utilitário global de formatação

Criar arquivo utils/textTransform.ts:

export function toUpperIfNotEmail(value: string, fieldName?: string): string {
  if (!value) return value;

  // se o campo for e-mail, não altera
  if (fieldName?.toLowerCase()?.includes("email")) {
    return value;
  }

  // mantém números e caracteres especiais normalmente, mas converte texto para MAIÚSCULO
  return value.toUpperCase();
}

2) Atualizar todos os componentes de input (TextInput)

Nos componentes que lidam com entrada de dados (TextField, Input, FormInput, ou o componente equivalente no projeto):

interceptar o onChangeText

aplicar o utilitário toUpperIfNotEmail

enviar o valor já convertido para o estado

Exemplo:

onChangeText={(text) => {
  const formatted = toUpperIfNotEmail(text, name);
  onChange(formatted);
}}

Em telas como:

Garantir que:
